# TFM-Gestion-del-Trafico-Aereo
Modelo Avanzado de Dispersión de Tráfico para la Optimización del Balance Demanda–Capacidad en la Gestión del Tráfico Aéreo
La gestión eficiente del tráfico aéreo en el espacio aéreo europeo requiere
herramientas avanzadas que permitan mantener el balance entre la demanda prevista de
vuelos y la capacidad declarada de los sectores. Tradicionalmente, el Gestor de Flujo (FMP)
utiliza medidas como regulaciones ATFM, cambios de sectorización, activación de
escenarios o imposición de restricciones verticales y horizontales. Estas medidas presentan,
sin embargo, gran rigidez, ya que se aplican a flujos completos de tráfico, sin discriminar
entre vuelos individuales.
Este Trabajo Fin de Máster desarrolla, analiza y evalúa un modelo avanzado de Traffic
Dispersion, un mecanismo que asigna a cada vuelo una trayectoria alternativa —con un
sector sequence distinto— con el fin de balancear dinámicamente las cargas entre sectores,
reduciendo congestiones sin necesidad de imponer regulaciones más invasivas. El trabajo
incluye: (1) una formalización matemática del problema como un modelo MILP; (2) el
diseño de heurísticas escalables adecuadas para entornos operacionales; (3) la creación de
un generador de alternativas verticales y horizontales; y (4) experimentos basados en datos
sintéticos y en casos realistas del espacio aéreo MUAC.
